# Arquitectura y Organización de Computadores

Repositorio de cursada enfocado en C, memoria, punteros, compilación y primeros pasos en assembler.

## Contenido

- Guía C básica
- Guía C avanzada
- Ejercicios de práctica
- Material de apoyo para la cursada

## Objetivo

Usar este repositorio para practicar programación en C, entender el modelo de memoria y conectar conceptos de alto nivel con su representación a bajo nivel.
